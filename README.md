## How to
* add the following files to your project:
  * _data/h5bp.html
  * _layouts/h5bp.html
  * _includes/analytics.html [optional]
* add the lines in _config.yml (if you don't already have them)
* in your layout <abbr title="YAML Front Matter">YFM</abbr> add layout: h5bp
* You're good to go..
